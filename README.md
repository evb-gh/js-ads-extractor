# js-ads-extractor

This project uses [Puppeteer API](https://pptr.dev/) to extract info from a job site.

## Installation

```bash
git clone https://github.com/evb-gh/js-ads-extractor.git
cd js-ads-extractor
npm install
```

## Usage

```bash
# after setting the desired query in pageScraper.js run
npm run start
```
